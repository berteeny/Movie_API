Marvel Movies API

This API holds data about every Marvel movie in "Release Order" (TV shows not included for now).

Description of Return Endpoint URL HTTP Method Type Request Body Data Format Response Body Data Format
Return a list of all movies /movies GET None A JSON object holding data about all movies
Return data about a single movie by title /movies/:title GET None A JSON object holding data about a single movie, ex:
{
_id: ObjectId('66529cb5816f90f3004bd561'),
title: 'Iron Man',
release: '2008',
director: {
name: 'Jon Favreau',
bio: 'Jonathan Kolia Favreau is an American filmmaker and actor, well known for his directorial and acting credits in multple films within the Marvel Cinematic Universe.',
birth: '1966-10-19',
death: ''
},
length: '2h 6m',
genre: {
name: 'comedy',
description: 'Comedy is a film genre that emphasizes humor. These films are designed to amuse audiences and make them laugh. Films in this genre typically have a happy ending.'
},
description: 'After being held captive in an Afghan cave, billionaire engineer Tony Stark creates a unique weaponized suit of armor to fight evil.' }
Return data about a single movie's genre by movie title /movies/genre/:genreName GET None A JSON object holding data about a genre, ex:
genre: {
name: 'comedy',
description: 'Comedy is a film genre that emphasizes humor. These films are designed to amuse audiences and make them laugh. Films in this genre typically have a happy ending.'
}
Return data about a director by name /movies/director/:directorName GET None A JSON object holding data about a director, ex:

director: {
name: 'Jon Favreau',
bio: 'Jonathan Kolia Favreau is an American filmmaker and actor, well known for his directorial and acting credits in multple films within the Marvel Cinematic Universe.',
birth: '1966-10-19',
death: ''
}
Allow existing users to log in /login POST Username and password params A message returning if there was a problem logging in
Allow new users to create a profile /users POST A JSON object that includes data required for the user profile, ex:
{
username: 'kramer',
password: 'ye-ow',
email: 'cosmo@mail.jk',
birthday: '1975-11-19'

} 		A JSON object holding data from the user profile that was created, along with a unique ObjectID, ex:
{
_id: ObjectId('6652aa94816f90f3004bd56d'),
username: 'kramer',
password: 'ye-ow',
email: 'cosmo@mail.jk',
birthday: '1975-11-19',
}
Allow users to update their user data /users/:username PUT A JSON object holding the user data you wish to be updated, ex:
{
"username": "joe updated name"
} 		A JSON object holding the user data, including updated data, ex:
{
_id: ObjectId('6652aa94816f90f3004bd56d'),
"username": "joe updated name",
"email": "joey@joetown.ca"
}
Allow users to add a movie to their list of favorites /users/:username/movies/:movieID POST None A JSON object holding the user data, including updated favMovie array, ex:
{
_id: ObjectId('6652aa94816f90f3004bd56d'),
"username": "joe updated name",
"email": "joey@joetown.ca"
"favMovies": ["6652a205816f90f3004bd567"]
}
Allow users to remove a movie from their list of favorites /users/:username/movies/:movieID DELETE None A JSON object holding the user data, including updated favMovie array, ex:
{
_id: ObjectId('6652aa94816f90f3004bd56d'),
"username": "joe updated name",
"email": "joey@joetown.ca"
"favMovies: []"
}
Allow existing users to delete their user profile /users/:username DELETE None A message returning if the user profile was successfully deleted
