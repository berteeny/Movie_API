Marvel Movies API

This API holds data about every Marvel movie in "Release Order" (TV shows not included for now).

Description of Return Endpoint URL HTTP Method Type Request Body Data Format Response Body Data Format
Return a list of all movies /movies GET None A JSON object holding data about all movies
Return data about a single movie by title /movies/:title GET None A JSON object holding data about a single movie, ex:
{
"title": "Iron Man",
"director":
{name: "Jon Favreau"},
"genre":
{name: "superhero movie",
description: "words" }
}
Return data about a single movie's genre by movie title /movies/genre/:genreName GET None A JSON object holding data about a genre, ex:
{
name: "superhero movie",
description: "words"
}
Return data about a director by name /movies/directors/:directorName GET None A JSON object holding data about a director, ex:

{name: "Jon Favreau"}
Allow new users to create a profile /users POST A JSON object that includes data required for the user profile, ex:
{
"name": "Joe",
"email": "joey@joetown.com"
} 		A JSON object holding data from the user profile that was created, along with a uuid, ex:
{
"name": "Joe",
"email": "joey@joetown.com"
"id": "b8d9e773-2b36-4967-83a5-af57301053ab"
}
Allow users to update their user name /users/:id PUT A JSON object holding the user data you wish to be updated, ex:
{
"name": "joe updated name"
} 		A JSON object holding the user data, including updated data, ex:
{
"id": 2,
"name": "joe updated name",
"favMovies": [
"Thor"
],
"email": "joey@joetown.ca"
}
Allow users to add a movie to their list of favorites /users/:id/:movieTitle POST None A message returning if the movie was added to user's favourites list
Allow users to remove a movie from their list of favorites /users/:id/:movieTitle DELETE None A message returning if the movie was removed from user's favourites list
Allow existing users to delete their user profile /users/:id DELETE None A message returning if the user profile was successfully deleted
